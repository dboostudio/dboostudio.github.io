- (심볼) : (특수키)
- ⌘ : Command key
- ⌃ : Control key
- ⌥ : Option or Alt key
- ⇧ : Shift key
- ⌫ : Delete key
- ↩ : Return or Enter key
- ↑ : Up arrow key
- ↓ : Down arrow key
- → : Right arrow key
- ← : Left arrow key

## IntelliJ Keymap
- 인텔리제이에서 단축키설정을 보려면
    Shift키를 세번연타(⇧ + ⇧ + ⇧)해서 Find EveryWhere창으로 들어간다.
    keymap으로 검색해서 들어가보면 단축키세팅들이 나오고, 단축키변경이 가능하다.

- 이전화면, 이후화면으로 이동
    ⌘ + [ : 이전화면으로 이동
    ⌘ + ] : 이후화면으로 이동

- 코드 자동 줄맞춤
    ⌘ + ⌥ + L

- 커서위치에서 세그먼트 잡기 (블록지정?)
    ⌥ + ↑
    여러번 연타하면 지정블록이 점점 넓어진다.

- 이전/다음 작업위치로 돌아가기
    ⌃ + ⌥ + ← or →

- 현재 지정된 문자열과 동일한 문자열 한번에 잡기
    ⌃ + G

- 멀티커서 위치잡기
    ⇧ + 마우스 왼쪽 버튼
    버튼을 클릭할때마다 멀티 커서 위치가 추가된다.
